Some of the features that I would like to bring to Wipefest include:

  • View a fight summary from the perspective of a single player, and see when they used their major cooldowns and defensives.
  • Compare two fights side by side.
  • Boss guides that dissect top guild's progress kills using Wipefest.
  • Support for more event types, such as "Interrupt", "Combat Resurrection", "Buff/debuff removed".
  • A summary for each fight with bullet points, such as:
    • (Kil'jaeden) "You gained Illidan's Sightless Gaze 5 seconds into Transition 2"
    • (Maiden) "60% of your raid gained 10 stacks of Creator's Grace"
    • (Harjatan) "You killed the boss after 2 Frosty Discharges"
    If I implement a backend, then this could be extended to also compare your summary to other guild's kills with similar item level:
    • (Maiden) "Similar guilds to you had 55% of their raid gain 10 stacks of Creator's Grace".

The configuration for what events to show in each fight is driven by JSON files from this public GitHub repository. These control what abilities, debuffs, spawns, and phases are shown in each fight, as well as how they are found within the combat log of the fight. I would really like to write some documentation for these files, so that anyone who wants to help can contribute by tweaking or adding new encounters. If we introduce player-specific views of fight summaries, it would be really helpful to have people who play each specialization write configuration files for the important abilities and buffs that should be shown in these views.
